What HQ Is All About

HQ is a Halloween-season music hub focused on concerts, playlists, and artists. It pulls together concerts, gossip, and visuals so everything lives in one place.

This site was built for a web design assignment using components like blog previews, cards, a Bootstrap navbar, and a concert carousel. The goal is to make something that looks like a real music site while practicing HTML, CSS, Flexbox, and a bit of Bootstrap.
